# FUTURE_DS_02

Churn and Customer Retention Rate Analysus Dashboard

This is a pOWERBI Dashboard visualised using PI Charts , Line Charts , Graphs , Funnel Metrics , etc. This provides key insights about :
1. Customer Gender
2. Churn Count
3. Tenure
4. Online Backup
5. Online Security

and followed by a few other metrics that contribute to gaining valuable insights.

TOOLS USED :
1. POWER-BI
2. Excel
3. Data Visualisation

TAKEAWAYS:
1. Money Transfer through Banks (i.e) Bank-Transfer turned out to be the most effectively and commonly used mode of payment over the other modes.
2. Male and Female Customers Count has been observed to be EVEN .
3. Method of Paper Billing is not used very often , Online services seem to dominate the Payment Statistics.
